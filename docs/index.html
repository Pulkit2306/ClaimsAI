Features
Enterprise-Grade Capabilities
Every feature mirrors real-world insurance industry requirements — from AI-assisted triage to regulatory audit trails.
auto_awesome

AI Claims Assistant

Natural language Q&A about claims and policies using RAG pipeline with pgvector embeddings and Claude AI.

gpp_bad

Fraud Detection

AI-driven risk scoring (0.0–1.0) with red flag identification. Every new claim is automatically analyzed via Kafka events.

manage_search

Semantic Search

Coveo-style multi-match search with fuzzy matching and relevance ranking across all claims and documents.

bolt

Event-Driven Pipeline

Apache Kafka powers real-time event processing — claim creation triggers AI analysis, search indexing, and audit logging.

verified_user

Claims Lifecycle

Full workflow from submission to settlement with role-based access for policyholders, adjusters, underwriters, and admins.

monitoring

Observability

Prometheus metrics, Grafana dashboards, Elasticsearch + Kibana for log analytics, and MongoDB audit trail.

Quick Start
Up and Running in 3 Commands
All you need is Docker Desktop installed. One command builds and starts everything.
# 1. Clone the repository
git clone https://github.com/Pulkit2306/ClaimsAI.git
cd ClaimsAI

# 2. Start everything (infrastructure + all services + frontend)
docker-compose up --build

# 3. Seed sample data (in a new terminal)
bash scripts/seed-data.sh

# Open http://localhost:4200 and login with:
# admin@claimsplatform.com / password123
